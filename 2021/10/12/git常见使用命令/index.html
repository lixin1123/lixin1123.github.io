git常见使用命令

  • 切换分支 git checkout feature-lx

  • 拉取别的分支 在dev上拉取lx的分支

    git checkout develop

    git merge feature-lx

  • 提交代码

    git stash

    git pull

    git stash pop

    git add -A

    git commit -m ‘’

    git push

  • 删除分支

    git branch -d feature-lx

  • 创建分支

    git checkout-b feature-file(名字以功能模块命名)
